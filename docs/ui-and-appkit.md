# UI、状态栏与 AppKit 细节

本文档说明状态栏、弹窗、设置页、快捷键和 SwiftUI/AppKit 桥接规则；修改 UI 或 macOS 平台行为时加载。

## 状态栏

核心文件：`SalaryDance/Views/FloatingPanelView.swift`。

| 状态 | 行为 |
|------|------|
| 实时金额关闭 | 只显示系统符号图标 |
| 实时金额开启 | 使用 SwiftUI `NSHostingView` 自定义内容 |
| 金额动画 | 支持滚动、跳动、关闭 |
| 状态栏金额颜色 | 默认白色，可配置 |
| 摸鱼状态图标 | 可单独开关，默认仅摸鱼中显示；关闭“仅摸鱼中显示”后常驻并随当前摸鱼状态切换图标 |

## 弹窗定位

弹窗使用 `NSPopover`，依附状态栏入口展示，点击弹窗外关闭。

## 弹窗内容

核心文件：

- `PopoverView.swift`
- `SalaryDisplayView.swift`
- `BalancedSalaryMetricGrid.swift`
- `WorkQuoteState.swift`
- `OffTaskTracker.swift`

规则：

- 弹窗宽度固定 280。
- 当前收入、今日剩余、状态、进度、各薪资指标、摸鱼状态、打工语录均可单独开关。
- 弹窗摸鱼状态内可单独开关本日、本周、本月/本周期、历史的摸鱼薪资和摸鱼时长；脱敏打开时摸鱼金额也需要脱敏，时长不脱敏。
- 真实弹窗和预览应优先共用同一 SwiftUI 组件；主薪资区复用 `SalaryDisplayView`，摸鱼状态区复用共享面板，避免按钮、脱敏小眼睛和指标网格出现两套实现。

## 设置页

核心文件：

- `SettingsView.swift`
- `SettingsSupportViews.swift`
- `PopoverPreviewView.swift`

规则：

- 设置页采用左侧分类 + 右侧配置面板。
- 展示页的弹窗内容按基础信息、薪资指标、摸鱼状态和其他分组，避免开关过多时变成单层列表。
- 展示页右侧弹窗预览保持在原右栏内，并与真实弹窗复用展示组件，避免预览和实际效果漂移。
- 薪资页包含基础薪资、计薪周期、月薪计薪方式、补贴列表和换算结果；补贴说明需要覆盖“是否进入实时收入”和“月薪汇总按原值计入”的差异。
- 补贴列表每项有开启/关闭开关；关闭后保留配置但不参与换算，用于对比。
- 摸鱼页展示当前状态、本日、本周、本周期/本月和历史；本周、本周期/本月必须展示起止日期。
- 本周期沿用用户配置的 `SalaryCycleMode`，自然月文案展示为本月，固定周期文案展示为本周期。
- 历史区展示历史累计和按年-月-日折叠的全部历史记录；当前状态区允许同步开启或结束摸鱼状态。
- 历史记录展开到单次记录后可编辑时间范围或删除记录，精确到秒；已结束记录只允许编辑开始和结束时间，进行中记录只允许编辑开始时间。
- 应用页提供导入导出：数据文件包含薪资、补贴、计薪规则、工作时间和摸鱼记录；配置文件包含展示、快捷键、颜色、刷新、开机启动等偏好。

## 快捷键

核心文件：

- `GlobalShortcutMonitor.swift`
- `ShortcutRecorderView.swift`
- `SalaryConfig.swift`

规则：

- 默认快捷键：Option + Command + Z。
- 快捷键动作是序列，每按一次执行下一项。
- 默认序列：打开状态栏实时显示、关闭状态栏实时显示。
- 可选动作：打开状态栏实时显示、关闭状态栏实时显示、脱敏打开窗口、不脱敏打开窗口、关闭窗口。
- 摸鱼切换是独立快捷键，默认 Option + Command + X，可单独启停和录制。
